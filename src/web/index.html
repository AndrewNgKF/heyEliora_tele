Rather host it yourself? Go for it.

Clone the repo, add your keys, and you have your own AI assistant.

# Clone it
$ git clone https://github.com/AndrewNgKF/heyEliora_tele.git
$ cd heyEliora_tele
 
# Set it up
$ bun install
$ cp .env.example .env
 
# Run it
$ bun run src/index.js

You'll need a Telegram Bot Token, an Anthropic API Key, and a Turso database.

Questions.

Is my data safe?

Self-host and we never see it. On the hosted version, you can wipe everything with /forget. The code is open source — audit exactly what gets stored.

Why Telegram?

Nobody needs another tab. Telegram is fast, on every device, and push notifications just work. A personal assistant should meet you where you are.

What AI model does she use?

Claude by Anthropic. Free uses Haiku (fast). Pro uses Sonnet (most capable). Self-hosters can swap the model.

How is this different from ChatGPT?

ChatGPT forgets you after every conversation. Eliora remembers your goals, tracks progress over weeks, learns your work style, and holds you accountable. She's not an everything-machine — she's your machine.

Will she support calendar, email, docs?

That's the plan. Reminders and scheduling are next, then document reading, then Google Calendar and Gmail. Roadmap is on GitHub.
